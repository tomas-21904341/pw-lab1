CSS

O CSS tal como o nome diz é um conjunto de regras que serve embelezar um site

tipos de seletores CSS
Elemento Usado? Explicação & Onde usado
Universal ✔️ Aplica o estilo a todos os elementos
Tipo ✔️ Aplica o estilo ao tipo de elemento
Classe ✔️ Aplica o estilo do nome de uma classe a utilizar
Id ✔️ Aplica o estilo a um id especifico
Atributos ✔️ Aplica o estilo a um atributo
Pseudo-classe ✔️ Aplica o estilo conforme o estado de um elemento
Pseudo-elemento ✔️ Aplica o estilo consoante em partes de elementos
combinadores de seletores
Seletor Usado? Explicação & Onde usado
Seletores descendentes ✔️ Aplica o estilo a todos os elementos que estão “dentro” de um elemento, utilizado no h1
Seletores filhos ❌ Aplica o estilo a todos os elementos que estão “logo dentro” de elemento
Seletores adjacentes ❌ Aplica o estilo a todos os elementos que estão colocados depois de um elemento
Seletores imediatamente adjacentes ❌ Aplica o estilo todos os elementos que estão colocados “imediatamente” depois de um elemento
Agrupados ✔️ Aplica o estilo nos elementos separados por virgula, utilizado com o p e o div
